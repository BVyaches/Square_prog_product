Программный продукт "Калькулятор корней"
Данный продукт написан на языке Python, в основу лёг мессенджер Telegram и его TelegramBotAPI.
Для запуска:
1. Установить Python
2. Командой pip install requirements.txt установить все нужные библиотеки
3. Запустить файл Bot.py
4. Подождать пару секунд (!!!) пока программа подключится к серверам
5. Готово!
P.S. Скорость работы программы напрямую зависит от скорости интернета, к которому подключено запускающее устройство.
Данные действия нужно выполнять один раз лишь со стороны администратора на одной машине. После запуска кода программа будет доступна абсолютно всем пользователям Telegram.
Если возникают вопросы по первичному запуску программы: vsbolshagin@edu.hse.ru или @Vyaches_s в Telegram. Если нужно - есть возможность деплоя на удаленный сервер программы для её работы 24/7 вне зависимости от состояния устройства администратора.

Для начала использования рядовому пользователю (после первичного запуска программы администратором):
1. Скачать Telegram на любое доступное устройство (Windows XP и выше, macOS 10.12 и выше, Linux, Android 4.1 и выше и iOS 9.0 и выше) ИЛИ перейти на веб-версию Telegram по ссылке https://web.telegram.org
2. Войти или зарегистрироваться в Telegram (интрукция доступна в приложениях и на сайте)
3. Перейти к диалогу с ботом по ссылке https://t.me/Calc_everything_bot или введя в поиск "@Calc_everything_bot".
4. Нажать кнопку "Старт"

Доступный функционал:
При отправке боту любого доступного выражения он сразу отправит извлеченный корень из него или сообщение, что произошла ошибка в формуле.
Для уточнения количества знаков после запятой нажать на кнопку на клавиатуре внизу чата "Посчитать с заданной точностью", далее следуйте инструкциям на экране.

Для смены языка нажмите на клавиатуре внизу чата кнопку с глобусом и рисунком настроек, далее следуйте инструкции. Программа написана с использованием GoogleTranslateAPI, поэтому поддерживает перевод на 106 языков мира.

Выражения, из который программа умеет извлекать корни:
1. Целые и вещественные числа
2. Числа в экспоненциальной записи
3. Комплексные и отрицательные числа
4. Выражения вида a**b, где a - любое число или переменная, b-четный показатель степени
5. Аналитические выражения, например формула полного квадрата или следствие основного тригонометрического тождества
6. Выражения вида a**b, где а - число, из которого нацело выделяется квадратный корень, b - любое число или переменная

Кроме того, в боте доступны команды для быстрого вызова нужных функций:
/start - начать общение с ботом
/help - узнать возможности бота
/commands - узнать список доступных команд
/cancel - отменить текущую операцию, вернуться в главное меню
/menu - вернуться в главное меню
Кроме того, все команды доступны в иконке слева от поля ввода сообщения.

------English-------
Software product "Root calculator"
This product is written in Python, based on the Telegram messenger and its TelegramBotAPI.
For start:
1. Install Python
2. Install all the necessary libraries with the pip install requirements.txt command
3. Run the Bot.py file
4. Wait a couple of seconds (!!!) until the program connects to the servers
5. Done!
P.S. The speed of the program directly depends on the speed of the Internet to which the launching device is connected.
These steps only need to be performed once by the administrator on one machine. After running the code, the program will be available to absolutely all Telegram users.
If you have questions about the initial launch of the program: vsbolshagin@edu.hse.ru or @Vyaches_s in Telegram.

To start using:
1. Download Telegram to any available device (Windows XP and above, macOS 10.12 and above, Linux, Android 4.1 and above and iOS 9.0 and above) OR go to the web version of Telegram at https://web.telegram.org
2. Log in or register in Telegram (instruction is available in applications and on the website)
3. Go to the dialogue with the bot using the link https://t.me/Calc_everything_bot or by entering "@Calc_everything_bot" in the search.
4. Press the "Start" button

Available functionality:
When sending any available expression to the bot, it will immediately send the extracted root from it or a message that an error has occurred in the formula.
To specify the number of decimal places, click on the button on the keyboard at the bottom of the chat "Calculate with a given accuracy", then follow the instructions on the screen.

To change the language, press the button with the globe and settings picture on the keyboard at the bottom of the chat, then follow the instructions. The program is written using GoogleTranslateAPI, therefore it supports translation into 106 languages ​​of the world.

Expressions from which the program can extract roots:
1. Whole and real numbers
2. Numbers in exponential notation
3. Complex and negative numbers
4. Expressions like a**b, where a is any number or variable, b is an even exponent
5. Analytic expressions, such as the full square formula or a consequence of the basic trigonometric identity
6. Expressions of the form a**b, where a is a number from which the square root is completely allocated, b is any number or variable

In addition, commands are available in the bot to quickly call the necessary functions:
/start - start chatting with the bot
/help - find out the capabilities of the bot
/commands - get a list of available commands
/cancel - cancel the current operation, return to the main menu
/menu - return to the main menu
In addition, all commands are available in the icon to the left of the message entry field.
